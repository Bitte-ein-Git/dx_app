A simple web app that displays the current online players/rooms for **Mario Kart Wii Deluxe X**, a Custom Track-Distribution for Mario Kart Wii.

**App available at: [https://dx.heyfordy.de](https://dx.heyfordy.de)**

## üöÄ Features

* Live view of all public and private online rooms.
* **Mii Rendering:** Automatic conversion of raw Mii data to Nintendo Studio images (via RC24).
* Local Favorites system
* Mobile and desktop-optimized layout.
* "Add to Home Screen" functionality for iOS and Android.

## ‚öôÔ∏è APIs Used

* **Live Data** | newwfc.xyz: [groups](https://stats.newwfc.xyz/api/groups?game=mariokartwii) & [json](https://stats.newwfc.xyz/api/json)
* **RC24 Mii Rendering** | [WiiLink/Riiconnect24 Mii Studio](https://miicontestp.wii.rc24.xyz/cgi-bin/studio.cgi)
